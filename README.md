# remote-control

## First. Clone repo

`git clone https://github.com/blinkoliver/remote-control.git`

## Second. Installation

`npm install`
or
`yarn`

## Third. Usage

`npm run start`
or
`yarn start`

- http server that gives frontend served @ `http://localhost:3000`
- server that host websocket run @ `http://localhost:8080`
